du skal skrive javascript der giver funktion til formen herunder.

Formen har følgende funktioner:

  • du kan indtaste euro rate i komma tal eks 1.4
  • du kan indtaste dollar rate i komma tal eks 1.4
  • du kan indtaste antal danske kroner

knappen "calculate" skal kalde javascript der kan udregne dollar kurs og sende resultatet til en view kode som du kan se herunder:

view funktion kunne se sådan her ud:

    
        function showResult(myEuroResult, myDollarResult, myDanishAmount) {
            const resultElement = document.getElementById('result');
            resultElement.innerHTML = myDanishAmount + ' danske kroner bliver til: ' + myEuroResult + ' euro og ' + myDollarResult + ' dollars';
        
        }
